# Aral Sea Environmental Change: GIS and Remote Sensing Analysis

## Overview

This repository contains GIS and remote sensing analyses examining environmental change in the Aral Sea region of Kazakhstan. The work focuses on monitoring coastline dynamics, identifying land cover changes, and evaluating environmental impacts associated with one of the world's most significant anthropogenic ecological disasters.

The Aral Sea has undergone dramatic transformation since the mid-20th century due to large-scale diversion of river water for irrigation. These changes have altered ecosystems, affected local communities, and reshaped the surrounding landscape. Through GIS techniques and satellite imagery analysis, this project investigates aspects of that transformation.

---

## Research Questions

* How has the coastline of the North Aral Sea changed over time?
* What evidence of environmental change can be detected using remote sensing techniques?
* How can supervised classification and change detection methods be used to quantify these changes?
* What insights can GIS provide regarding long-term environmental processes in the Aral Sea region?

---

## Methods

The analyses presented in this repository employed the following approaches:

* Remote sensing using satellite imagery
* ArcGIS Pro workflows
* Supervised image classification
* Training sample development
* Water versus non-water land cover classification
* Change detection analysis
* Spatial visualization and cartographic presentation
